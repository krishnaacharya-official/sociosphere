### Features:
- Responsive design for both mobile and web
- User can post with attachment
- User can make friends
- User can like
- User can comment on post


#### How to run ?

- Clone the repository
- Open 2 terminal 
- Type in the following command in 1st terminal
   ```
   cd server
   npm install
   npm start
   ```
- Type in the following command in 1st terminal
   ```
   cd client
   npm install
   npm start
   ```
